VectorForge

A production-style Retrieval-Augmented Generation system built with FAISS, hybrid reranking, and a fully local LLM pipeline.

Semantic Retrieval

Dense vector search using SentenceTransformers and FAISS for high dimensional semantic retrieval.

Hybrid Retrieval

Dense + BM25 hybrid retrieval with cross-encoder reranking for improved ranking precision.

Local LLM

Grounded answer generation using Llama3 via Ollama with inline citations and evidence blocks.

Retrieval Metrics

Measured retrieval quality using Precision@K, MRR, and end to end latency instrumentation.
